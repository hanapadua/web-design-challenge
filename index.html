WEATHER PLOT SUMMARY
(Latitude vs. X)


Welcome! This website is an analysis of how weather across over 500 cities all over the world changes the closer these cities are to the equator. This analysis was done using OpenWeatherMap's API and used to create a a representative model of weather across world cities.


Matplotlib was used to create a series of plots to showcase the correlations regarding several variables of weather vs. latitude. The relationships that were analyzed included:

Temperature (F) vs. Latitude,
Humidity (%) vs. Latitude,
Cloudiness (%) vs. Latitude,
and Wind Speed (mph) vs. Latitude.


There are data and visualizations part of the analysis, including explanations and descriptions of trends and correlations.
